# schedar-agents

Docker sandbox template for [Claude Code](https://docs.claude.com/en/docs/claude-code), built for use with [VSHN AppCat "AI with AppCat"](https://kb.vshn.ch/app-catalog/how-tos/ai-with-appcat.html).

Extends `docker/sandbox-templates:claude-code` with a preinstalled plugin set (see [Dockerfile](Dockerfile)).
